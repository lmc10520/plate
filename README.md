**Web App 功能摘要**

 - Responsive Design (響應式設計) 確保手機、平板與電腦上體驗一致。
 - 透過 Python 程式控制，抓取原停車場查詢停車位功能後進行統計，對停車場內停泊車輛數進行統計後的呈現。
 - 透過圓餅圖及折線圖，將目前即時資訊以可視化方式顯示。統計的數據儲存於 json ，可透過日期篩選功能進行趨勢分析。
 - 提供簡易輸入欄位，輸入車牌後得到停車格格號，即可使用其資訊返回地圖模式，進行行徑動線的引導。
 - 所有資訊僅記錄去識別化資訊，不會對可識別資訊進行紀錄與進一步的分析。

**解決問題**
 - 人潮可預期的湧現前，進行趨勢的監控。
 - 停車場爆滿時，進行導引人力的調撥依據，以提供較佳的指引服務及人力安排。
 - 回顧趨勢，以進行優化之參考依據。
